# JavaScript CLI

[[Sự khác nhau giữu npm vs yarn]]

`yarn create react-app project-name` 

#### Start the development server
`yarn start`

#### Đóng gói App từ cho production.
`yarn build`

#### Chạy test
`yarn test`
