# Logarithmic time – O (log n)

Thuật toán có độ phức tạp là Logarithmic time khi nó giảm số lần chạy so với số lượng đầu vào của input data trong mỗi lần chạy. Binary Trees, Binary Search là 1 ví dụ cụ thể.
